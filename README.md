# My webpage Project

## Project Overview
This project is a simple cafeteria webpage created using HTML and CSS.  
The webpage provides information about the cafeteria, including the menu, reservation and online delivery options.

## Steps Completed

1. Created an HTML file (`index.html`) with a semantic structure including header, main, and footer.
2. Added a table to display sample data and a form to collect user input.
3. Created a CSS file (`styles.css`) to style the page layout, table, and text using CSS selectors.
4. Made multiple Git commits with descriptive messages explaining each change.
5. Pushed the project to a public GitHub repository.
6. Deployed the webpage using GitHub Pages.

## Technologies Used
- HTML
- CSS
- Git & GitHub

## Live Website
The webpage is deployed using GitHub Pages.
